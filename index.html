About
JSPlay is a simple JavaScript playground that allows you to write and execute JavaScript code in a single. It is a web-based application that runs in your browser without any installation. This Editor support for version Javascript Popular and default EcmaScript 2015 (ES6).
If you see a bug or want to request a feature, please create an issue on GitHub and I will try to fix it as soon as possible. Also feel free to fork this project and send a pull request.
